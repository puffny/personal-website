# 个人作品集网站 PRD

Version 1.0

## 1. 项目名称

梁慧锋个人作品集网站

## 2. 网站定位

打造一套面向招聘方、面试官及潜在合作伙伴的个人品牌网站。

网站不以展示项目数量为目的，而是通过职业经历、核心能力、项目案例与职业思考，完整呈现个人设计价值。

## 3. 用户群体

### 主要用户

- HR
- 招聘经理
- 产品负责人
- 设计负责人

### 次要用户

- 合作伙伴
- 客户
- 设计同行

## 4. 核心目标

用户进入网站后能够快速获得以下信息：

- 我是谁：12年 UI/UX 设计经验。
- 我做过什么：企业数字化平台、政务产品、移动端产品、品牌与视觉设计。
- 我擅长什么：复杂系统设计、用户体验优化、视觉与交互表达。
- 我的思考方式：设计价值、AI 认知、职业发展方向。
- 如何联系我：邮箱、手机号、微信。

### 01 天目数智

定位：复杂系统设计案例。

项目背景：

面向企业数字化运营场景，业务模块众多、角色复杂，存在信息架构混乱、设计规范缺失、协作效率低等问题。

我的职责：

- 信息架构梳理
- 设计系统搭建
- 核心业务流程设计
- 设计规范制定与落地

设计过程：

- 梳理业务流程与角色权限关系。
- 重构导航与信息层级。
- 建立组件库与设计规范。
- 推动设计与开发协同机制。

标准化成果：

- 核心页面设计效率提升约 40%。
- 设计组件复用率提升至 80% 以上。
- 新功能设计交付周期缩短约 30%。
- 跨团队协作沟通成本显著降低。

项目价值：

通过系统化设计方法提升复杂业务场景下的产品一致性与团队协作效率，为后续产品扩展提供可持续设计基础。

### 02 浙里报账

定位：用户体验优化案例。

项目背景：

政务报账流程涉及多个审批节点，用户操作路径长、理解成本高，影响整体办理效率。

我的职责：

- 用户研究
- 流程诊断与优化
- 交互设计
- 可用性验证

设计过程：

- 用户访谈与问题收集。
- 绘制现状流程图。
- 优化关键任务路径。
- 进行体验测试与迭代验证。

标准化成果：

- 核心报账流程步骤减少约 35%。
- 用户任务完成时间缩短约 45%。
- 流程一次性提交成功率提升约 28%。
- 用户满意度提升至 90% 以上。

项目价值：

有效降低用户学习成本与操作负担，在满足业务规范要求的同时提升整体使用体验。

### 03 森本新闻

定位：视觉与交互表达案例。

项目背景：

品牌传播内容同质化严重，需要通过视觉升级与交互创新提升用户关注度与品牌识别度。

我的职责：

- 品牌视觉设计
- 活动专题设计
- 交互动效设计
- 创意方案输出

设计过程：

- 品牌调性分析。
- 视觉概念探索。
- 页面与动效设计。
- 多轮方案优化与落地。

标准化成果：

- 活动页面停留时长提升约 32%。
- 内容点击转化率提升约 25%。
- 品牌视觉一致性显著增强。
- 用户互动参与率提升约 20%。

项目价值：

通过视觉与交互协同设计提升品牌传播效果，增强用户记忆点与内容吸引力。
